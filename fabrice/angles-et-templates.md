## Par type de post Reddit/FB

### "Mon site est lent" / "Slow store" / "Page speed"
FRAMEWORK : Diagnostic technique en 3 couches. 1) Apps installées (combien ?). 2) Images (format, taille). 3) Theme (code bloated ?). Donner le fix le plus impactant en premier.

### "Quel outil pour X ?" / "Best tool for..."
FRAMEWORK : Comparatif technique. Ce que chaque outil fait SOUS LE CAPOT. Pas juste les features marketing. Recommandation basée sur le use case technique.

### "Comment automatiser X ?" / "Workflow automation"
FRAMEWORK : Step-by-step simplifié. "Here's exactly what I set up:" puis les étapes. Montrer que c'est faisable, pas juste théorique.

### "Mon store/page/produit ne marche pas" (diagnostic)
FRAMEWORK : Questions de debugging. 3 questions ciblées pour identifier le vrai problème technique. Puis la solution une fois le diagnostic fait.

### "How to repurpose content" / "Too many tools" (creators M2)
FRAMEWORK : Builder story. "I built a workflow that does this in 3 steps instead of 7 tools." Montrer le process simplifié.

### "SEO / indexing issues" (e-com)
FRAMEWORK : Pourquoi technique. Expliquer pourquoi Shopify a des limitations SEO structurelles et quels workarounds existent. Concret, pas théorique.
