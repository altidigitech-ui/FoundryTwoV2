# GUIDE PUBLICATION — Comment demander à Claude

**Usage :** Guide pour F. Comment formuler les demandes au projet Claude "Fabrice" pour des posts de qualité.

---

## 1. LE FLOW

```
1. Vendredi soir — Réunion R+F : F fournit ses inputs techniques.
2. F rédige à SON rythme (samedi ou au fil de la semaine).
3. Pour chaque post : plateforme + type + notes techniques brutes + vertical + mots bruts.
4. Claude génère le post dans la voix F.
5. F valide, ajuste, programme.
6. En semaine — Zéro rédaction. Publication des posts programmés.
```
