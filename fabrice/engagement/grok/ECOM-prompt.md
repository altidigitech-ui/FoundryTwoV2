# PROMPT GROK — Recherche posts à engager e-com (angle technique)

**Usage :** Prompt FIXE à copier-coller dans Grok. F ouvre Grok, colle ce prompt, Grok retourne les posts pertinents pour l'engagement. Dédié aux conversations e-com avec un angle TECHNIQUE.

---

## PROMPT À COPIER

```
Search Twitter for "shopify" in the Latest tab. Show me 15 recent posts (last 24-48 hours) in English that talk about store speed, apps, themes, checkout, page load, mobile performance, or any technical aspect of running a Shopify store.

For each post give me:
1. URL
2. Author + follower count
3. Likes and replies count
4. One line summary of what they're saying

Include everything: complaints, tips, questions, discussions, opinions. The only things I don't want are job postings, pure ads, and posts in other languages.

Do NOT exclude posts because they mix technical and business angles. Do NOT exclude advice posts. Include them all.
```

---

## NOTES

- Les résultats sont organisés en 2 tableaux : Table 1 (visibilité, gros comptes 10K-100K) et Table 2 (crédibilité, experts niche 1K-10K).
- Ce prompt cible les conversations TECHNIQUES e-com. Différence avec CREATOR-prompt engagement : ici on cherche des conversations sur Shopify performance, apps, theme code — pas sur le workflow des creators.
- Différence avec R ECOM-prompt engagement : R engage sur les sujets BUSINESS (conversion, pricing, chargebacks). F engage sur les sujets TECHNIQUES (speed, apps, code).
