## Rôle

Ce fichier documente les règles de recherche Grok pour le contenu du compte studio @foundrytwo. R utilise le prompt STUDIO-prompt.md pour trouver des sujets tendance cross-vertical (e-com, agences, creators) avant le batch samedi.

## Différence avec les prompts R et F

| Prompt | Cherche | Pour qui |
|--------|---------|---------|
| R ECOM-prompt | Tendances e-com & agences (angle business) | Posts R perso |
| R AGENCY-prompt | Tendances agences (angle business) | Posts R perso |
| F CREATOR-prompt | Tendances creators & e-com (angle technique) | Posts F perso |
| F ECOM-prompt | Tendances e-com (angle technique) | Posts F perso |
| **F2 STUDIO-prompt** | **Tendances cross-vertical (3 verticals + build in public)** | **Posts F2 studio** |

## Ce que R fait avec les résultats

R copie les sujets tendance Grok → les donne au projet Claude "FoundryTwo" → Claude rédige les posts F2 dans la voix studio ("we", données terrain, build in public).

## Squelette éditorial F2

Les sujets trouvés alimentent le squelette 5j/sem :
- Lundi : Metrics portfolio
- Mardi : Hot take vertical (le sujet tendance de la semaine)
- Mercredi : Data thread (données terrain CDV × tendance)
- Jeudi : Question communauté (pour merchants/freelancers/creators)
- Vendredi : Build in public recap
